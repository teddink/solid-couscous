Description

This sample shows how to create a gesture object and assign a pointer to it. It also shows how to create the basic event handlers that you'll use to manage touch interaction. Touch the box and slide it around the screen. Tap the box to change color. Windows provides inertia of an object after you let go.

Code

      //  Get the div object, and assign the pointerdown event to it
      var testbox = document.getElementById("divBack");
      testbox.addEventListener("pointerdown", onDivPointerDown, true);

      function onDivPointerDown(e) {
      //  Assign the current pointer to the gesture object
      this.gestureObject.addPointer(e.pointerId);
    }
